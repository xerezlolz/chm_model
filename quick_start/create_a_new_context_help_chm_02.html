快速上手 - 用EASY CHM制作上下文相关的CHM帮助文件

EASY CHM不仅可以制作CHM格式的电子书、产品说明书等，同时也可以非常简单地制作供程序调用的支持上下文相关的CHM帮助文件，制作的具体步骤与"快速上手 - 用EASY CHM制作一个新的CHM文件"里的步骤一样.

因为Easy CHM默认的设置就是自动生成上下文相关的CHM帮助文件, 也就是说, 所有的CONTEXT-ID都被自动生成了. 

如果您不是程序员或者您不需要编写支持上下文相关的帮助文件, 在"Easy CHM 工程设置 - HtmlAPI对话框"对话框里不要选中 "编译 Html API 信息". 

如果您是程序员或者您需要编写支持上下文相关的帮助文件, 在生成CHM后,您可以在"ecAlias.h"和"ecMap.h"这两个文件里找到全部的CONTEXT-ID定义 (这两个头文件是Easy CHM自动生成的, 您可以在工程文件所在的根目录下找到他们. 这两个文件不需要对外发布,但您在程序里要用到里面定义的CONTEXT-ID.), 在"ecMap.h"里您可以看到类似下面的CONTEXT-ID定义:

#define IDH_feature 1118 ;Key Features 
#define IDH_About 1119 ;What is Easy CHM? 

这里的1118和1119都是CONTEXT-ID.
