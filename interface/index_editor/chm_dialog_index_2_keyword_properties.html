Easy CHM 界面- 索引编辑区 - 关键词属性对话框

当新建或打开一个工程后，您可以通过用鼠标点击任意的索引项，然后在索引编辑区的右键弹出菜单里选取"关键词属性..."或者您也可以通过选取程序索引编辑区的"关键词属性"按钮将看到这个对话框。关于这个对话框的使用请看下面的具体描述.　

　

标题:
在这里您可以编辑索引项（即关键词）的标题 - 列表框里备选的是链接文件的完全文件名或文件名，您也可以选择它们作为该目录项的标题。

文件名:
修改当前索引项（即关键词）链接的文件。

书签跳转:
如果当前链接文件里存在书签的定义，那您在这里也可以选择链接到这个文件的某个书签的位置 - 需要以#开始。
