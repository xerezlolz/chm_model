Easy CHM 界面 - 索引编辑区 - 右键弹出菜单



添加关键词:
弹出添加目录项对话框并允许您向CHM索引列表里增加新关键词。

添加子关键词:
弹出添加目录项对话框并允许您向CHM索引列表里增加新的子关键词。

导入:
向CHM索引列表里增加文件, 参见右键弹出菜单 - 导入。

输出:
输出CHM索引列表内容, 参见右键弹出菜单 - 导出。

移动:
上下左右移动CHM索引列表里选中的项目, 支持多选操作, 请参见右键弹出菜单 - 移动。

删除:
删除CHM索引列表里选中的项目。

替换:
替换CHM索引列表里项目的文字。

排序:
对CHM目录索引列表里的关键词进行排序, 参见: 排序。

全部展开:
全部展开CHM索引列表里的所有关键词 - 特别是多级目录时。

全部收缩:
全部展开CHM索引列表里的所有关键词。

编辑:
编辑CHM索引列表里选中的关键词的标题。

目录项属性:
显示CHM索引列表里选中的关键词的属性。
